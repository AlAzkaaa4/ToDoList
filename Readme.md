Aplikasi To Do List ini merupakan aplikasi berbasis Laravel yang digunakan untuk mencatat dan mengelola daftar tugas. Aplikasi ini dibuat untuk menerapkan konsep CRUD (Create, Read, Update, Delete) serta memahami penerapan MVC (Modelâ€“Viewâ€“Controller) pada Laravel.

Fitur yang tersedia dalam aplikasi ini meliputi penambahan tugas, penambahan deadline, catatan tugas, pengeditan tugas, penandaan tugas selesai atau belum selesai, penyimpanan tanggal selesai secara otomatis, serta penghapusan tugas. Tampilan aplikasi dirancang dengan layout dua kolom, yaitu form tambah tugas di sebelah kiri dan daftar tugas berbentuk card di sebelah kanan, sehingga terlihat rapi dan mudah digunakan.

Aplikasi ini dibuat sebagai bagian dari tugas perkuliahan dan dapat dikembangkan lebih lanjut sesuai kebutuhan.
